🎬 CDN Detector Manifest 功能測試

這是一個簡化的測試頁面，用於驗證任務 22.1 的 Manifest 攔截與解析功能。

📋 測試步驟

步驟 1：載入擴充功能
  1. 開啟 Chrome 擴充功能管理頁面：chrome://extensions/
  2. 開啟「開發人員模式」
  3. 點擊「載入未封裝項目」並選擇 cdn-tools 資料夾
  4. 確認 CDN Detector 出現在列表中並已啟用
步驟 2：測試 Manifest 檔案載入
步驟 3：檢查 Console 日誌
  1. 按 F12 開啟 DevTools
  2. 切換到 Console 標籤
  3. 尋找 🎬 Manifest file detected 訊息
  4. 確認看到解析完成的訊息
步驟 4：檢查擴充功能 Popup
  1. 點擊工具列中的 CDN Detector 圖示
  2. 查看是否顯示 Manifest 相關資訊
  3. 確認與 CDN 檢測功能並存

🔍 預期結果

✅ 成功標準：
  • Console 顯示 "🎬 Manifest file detected" 訊息
  • 顯示 "✅ DASH/HLS parsing completed" 訊息
  • 顯示解析的串流資訊（解析度、頻寬等）
  • DRM 保護狀態檢測正確
  • 沒有 JavaScript 錯誤
🔧 除錯提示：
  • 如果沒有看到訊息，檢查擴充功能是否正確載入
  • 確認 Console 沒有被過濾器隱藏訊息
  • 檢查網路連線是否正常
  • 某些 CDN 可能有 CORS 限制

🌐 真實網站測試

完成上述測試後，請訪問以下網站進行真實環境測試：

  • YouTube：開啟任何影片，檢查 Console
  • Twitch：觀看直播串流
  • Vimeo：播放高畫質影片

在這些網站上，您應該能在 Console 中看到 manifest 檔案被檢測和解析。
