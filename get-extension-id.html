🔧 CDN Detector 擴展 ID 工具

⚠️ 注意：此頁面需要在安裝了 CDN Detector 擴展的 Chrome 瀏覽器中開啟。

📋 手動獲取步驟

  1. 開啟新標籤頁，前往：chrome://extensions/
  2. 開啟右上角的「開發者模式」開關
  3. 找到「CDN Detector」擴展卡片
  4. 在擴展卡片上會顯示 ID（32位字母數字組合）
  5. 複製該 ID 並貼到測試頁面的輸入框中

🔗 快速連結

📝 範例擴展 ID 格式

Chrome 擴展 ID 通常是 32 位的小寫字母和數字組合，例如：

abcdefghijklmnopqrstuvwxyz123456

您的 CDN Detector 擴展 ID 會是類似的格式。

❓ 常見問題

  • 找不到 CDN Detector？ 確認擴展已安裝且已啟用
  • 沒有看到 ID？ 確認已開啟「開發者模式」
  • 連接測試失敗？ 檢查 ID 是否正確複製，沒有多餘空格
