⚠️ 重要使用說明

此調試頁面必須在 Chrome 擴充功能環境中開啟才能正常工作。

正確開啟方式：

  1. 開啟 Chrome 擴充功能管理頁面：chrome://extensions/
  2. 找到 "CDN Detector" 擴充功能並複製其 ID
  3. 在網址列輸入：chrome-extension://[擴充功能ID]/debug-video.html
  4. 或直接點擊擴充功能詳情頁面中的相關連結

如果直接開啟 HTML 檔案將無法使用 Chrome 擴充功能 API！
