🎬 串流平台測試工具

這個工具用於測試不同串流平台的媒體片段監控功能。

🧪 測試步驟

  1. 開啟串流平台：在新標籤頁開啟您要測試的平台
  2. 播放影片：開始播放任何影片
  3. 等待載入：讓影片播放至少 30-60 秒
  4. 檢查資料：回到這個頁面點擊下方按鈕檢查

📊 監控功能

🎯 建議測試平台

  • GagaOOLala - www.gagaoolala.com
  • Netflix - 使用 DASH 串流
  • Disney+ - 使用 DASH 串流
  • Twitch - 使用 HLS 串流
  • YouTube - 使用 DASH 串流（參考用）

🔍 串流技術檢測

不同平台使用不同的串流技術：

  • DASH：產生 .m4s 檔案
  • HLS：產生 .ts 檔案
  • Progressive Download：單一 .mp4 檔案（不會產生片段）
  • WebRTC：即時串流（不會產生檔案片段）
